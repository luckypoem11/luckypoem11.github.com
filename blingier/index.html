blingier is a URL shortener engine powered entirely by Java and designed to run anywhere (with minimum tweaks) that allows anyone to host their very own service.

Inspired by the open source YOURLS and based solely on the blingaer project, blingier attempts to provide all the features and power of blingaer without the requirement of the Google App Engine.

blingier will remain in the planning stage until development on blingaer is complete and a stable version is publicly released.
